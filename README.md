📊 Customer Behavior Analysis

📌 Project Overview

This project analyzes customer shopping behavior using data analysis techniques. The goal is to identify patterns, trends, and insights that help businesses make better decisions.

---

🚀 Features

- Data cleaning and preprocessing using Python
- Exploratory Data Analysis (EDA)
- SQL queries for data extraction
- Interactive dashboard using Power BI
- Insights on customer purchasing behavior

---

🛠️ Tech Stack

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI
- Jupyter Notebook

---

📂 Project Files

- "customer_shopping_behavior.csv" → Dataset
- "customer_behavior_Analysis.ipynb" → Data analysis in Python
- "customer_behavior_Analysis.sql" → SQL queries
- "Customer_Behavior_Dashboard.pbix" → Power BI dashboard

---

📊 Key Insights

- Identified top-selling product categories
- Analyzed customer purchase patterns
- Found peak shopping times
- Customer segmentation based on behavior

---

▶️ How to Run

1. Download the dataset
2. Open Jupyter Notebook
3. Run "customer_behavior_Analysis.ipynb"
4. Open Power BI file for dashboard

---

🎯 Conclusion

This project helps in understanding customer behavior and improving business strategies through data-driven decisions.
